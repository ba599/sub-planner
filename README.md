# 이벤트 플래너

게임 이벤트에서 **상점 아이템을 다 사려면 어느 스테이지를 몇 번 돌아야 가장 효율적인지** 자동으로 알려주는 도구입니다. https://justin163.com/planner/events 이 최신화 되어있지 않을 때, 임시로 사용하기 위해 만들어졌습니다.

## 사용 방법 (4단계)

### 1. 재화 등록하기

이벤트에서 모으는 재화(코인, 토큰, 조각 등)를 모두 적어주세요.
재화에 **획득량 보너스**가 붙어 있다면 보너스 % 도 함께 입력합니다. (예: 50% 보너스가 있다면 `50` 입력)

> 보너스가 여러 개로 쪼개져 있을 때는 **`+` 로 이어서** 입력해도 됩니다. 예를 들어 파티에 25% 캐릭터 1명과 15% 캐릭터 5명이 있다면 `25+15+15+15+15+15` 처럼 적으면 합산(=100)되어 계산됩니다. 인게임에서 확인한 수치를 그대로 옮겨 적기 편합니다.

### 2. 스테이지 드랍 입력하기

이벤트 스테이지는 **12개**, 4개씩 묶여 **3그룹**으로 나뉘어 있습니다.
각 스테이지에서 재화가 몇 개씩 떨어지는지 표에 채워 넣어주세요.

> 그룹 단위로 체크박스를 끌 수 있습니다. "맨 마지막 그룹은 아직 못 깼어요" 같은 상황이면 해당 그룹을 꺼두면 그 스테이지는 계산에서 빠집니다.

### 3. 사고 싶은 상점 아이템 입력하기

상점에서 사고 싶은 아이템의 **가격**, **살 개수**, **어떤 재화로 사는지**를 적어주세요.
아이템 이름 칸에서 글자를 입력하면 **자동완성**이 떠서 자주 쓰는 아이템을 빠르게 채울 수 있습니다.

### 4. 결과 확인하기

맨 위 **"보유량 & 결과"** 칸에서

- 어느 스테이지를 **몇 번 돌면 되는지**
- 다 합쳐서 **AP가 얼마나 드는지**
- 지금 가진 재화 + 앞으로 얻을 재화 vs 필요한 재화 (남는지/모자라는지)

가 자동으로 나옵니다.
이미 가진 양으로 충분하면 **"이미 충분합니다 ✅"** 라고 알려줍니다.

## 알아두면 좋은 팁

- **이미 가진 재화도 입력하세요.** 보유량을 적어두면 그만큼 빼고 계산해 줍니다.
- **자동 저장됩니다.** 입력한 내용은 브라우저에 자동으로 저장돼서, 페이지를 닫았다 다시 열어도 그대로 있습니다.
- **다른 기기로 옮기고 싶다면** 우측 상단 💾(저장) 버튼으로 파일을 저장한 뒤, 다른 기기에서 📂(불러오기)로 열면 됩니다. 친구한테 공유할 때도 같은 방법을 쓰면 됩니다.
- **"목표 달성 불가능"이 뜬다면** 입력한 드랍량으로는 그 아이템을 살 수 없다는 뜻입니다. 드랍 입력이 빠진 스테이지가 있는지, 비활성화한 그룹이 너무 많지 않은지 확인해 주세요.
- **자동완성 프리셋의 개수는 이벤트마다 다릅니다.** 자동완성이 채워주는 가격/개수는 참고용이니, 인게임 상점 화면과 한 번 더 비교해 주세요.
